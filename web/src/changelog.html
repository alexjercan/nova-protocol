Changelog

Every release of Nova Protocol

The illustrated companion to CHANGELOG.md: one page per release, grouped by what changed and why it matters, with migration notes when a format breaks. For the story behind a release, follow its link to the devlog.

    2026-07-16 // v0.6.0

    v0.6.0 - Mods over the wire

    A static mod portal and an in-game Explore online tab install and update mods over the wire, dependencies resolve end to end, a Scenarios picker lands on the main menu, and particles return to the web on WebGPU.

    A full web wiki and a trimmed first-scenario tutorial, rounded-out gamepad bindings, and a testable numbered-example curriculum gating CI.

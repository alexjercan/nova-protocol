# v0.3.1 - New engine, new home for the docs

A maintenance release that pulls Nova Protocol up to Bevy 0.19 and gives the
project a proper documentation home. No new gameplay to speak of - this is the
groundwork that keeps the game building on a current engine and makes the
codebase legible to newcomers.

## Bevy 0.19

The whole stack moves up to **Bevy 0.19**, dragging the physics and randomness
crates along with it: avian3d 0.7, bevy_rand 0.15, bevy_enhanced_input 0.26, and
rand 0.10. It is a mechanical upgrade with no player-facing behaviour change, but
it keeps Nova on a supported engine line.

## A docs home

The vendored `bevy_common_systems` copy is gone, now pulled in as a proper git
dependency instead of living in-tree. Alongside it lands an `AGENTS.md` and a new
`docs/` folder covering architecture, the scenario system, sections, development,
and migration notes - the first real home for how the project fits together.

## Rendering

A small addition on the render side: a **post-processing camera component**, the
seed of a proper post-processing pass.

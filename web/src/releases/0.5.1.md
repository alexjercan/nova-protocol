# v0.5.1 - Web build shakeout

A tiny patch that clears two crashes unique to the shipped WASM build. Both only
bit the browser when you started a run - New Game from the menu or Play from the
editor - and never touched native. If you play on the web, this is the one to
grab.

## Fixes

The web build no longer quits with a fatal render validation error on New Game
or editor Play: the target inset was carrying a `view_formats` override that
WebGL2 does not support for a plain sRGB target, so it now drops that override.

The skybox cubemap's `.meta` loader settings actually apply in the shipped app
again. `AssetMetaCheck::Never` had silently ignored them, which resurrected an
oversized-upload race that crashes GPUs with a 16384 texture limit.

The terse, full-detail log for this release lives in
[CHANGELOG.md](https://github.com/alexjercan/nova-protocol/blob/master/CHANGELOG.md).

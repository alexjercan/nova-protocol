# v0.5.2 - Learn the ropes on the web

A patch that turns the site into a place you can actually learn from: a full
wiki lands alongside a tutorial trimmed to first-scenario onboarding and two new
devlogs. Gamepad bindings get rounded out, the example suite becomes a testable
curriculum with CI smoke blocking again, and a handful of audio and teardown
bugs get fixed.

## A wiki and a tutorial on the web

The site grew a full **wiki** - gameplay, ship sections, keybinds, world, and
meta pages - so the game's systems finally have a written home. The tutorial was
trimmed down to first-scenario onboarding, and two new devlogs went up.

## Controls

Gamepad bindings were rounded out: ORBIT now maps to South, scenario-advance
confirm to DPadDown, and HUD cycle, pause, and back-to-editor each gained a
button.

## A testable example curriculum

The examples are now a testable curriculum: twelve numbered examples, from
controller PD through the boot flow, each self-driving under `BCS_AUTOPILOT`
with behavior assertions and a completion backstop. CI's example smoke suite is
BLOCKING again now that the GitHub-runner-only taffy panic is gone, so the
12-example suite gates every push. Smoke now fails on ANY logged command error,
closing the gap where handled remove/despawn warnings sailed past the panic
gate, and the weapon test ranges fire again after the safety had silently
disarmed both.

## Fixes

- Thruster hum now attenuates with distance per ship, so another ship's or
  torpedo's burn no longer plays at full volume from anywhere - your own ship
  stays exempt.
- Scenario teardown no longer logs an "Entity despawned" command error when a
  ship despawns with autopilot engaged: the telemetry-cleanup observer uses the
  fallible remove variant, pinned by regression tests.
- The debug inspector's window-camera placement fix moved upstream into
  bevy-common-systems, deleting nova's local workaround.

Full, terse detail for every one of these lives in
[CHANGELOG.md](https://github.com/alexjercan/nova-protocol/blob/master/CHANGELOG.md).

# v0.2.1 - Notes for the modders

A small release aimed squarely at anyone writing mods: the modding surface finally
gets real documentation and worked examples, and the event system underneath it
gets a cleanup pass. Nothing player-facing changes here - this is about making the
modding path easier to follow and the events it rides on easier to reason about.

## Modding docs and examples

The modding capabilities landed in v0.2.0 now come with **documentation and
examples** to match. Instead of reading the source to figure out how to hook into
the game, you get written guidance and worked samples to start from.

## Event system refactor

The **event system** got a refactor - the same events and queue introduced in
v0.2.0, reworked to be cleaner underneath. It stays a groundwork change with no
player-facing effect.

Full, terse detail for this release lives in
[CHANGELOG.md](https://github.com/alexjercan/nova-protocol/blob/master/CHANGELOG.md).

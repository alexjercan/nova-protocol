Devlog

Notes from building Nova Protocol

  • 2026-07-13  //  v0.5.0

    Devlog #5: radar locking, a tutorial, and a home on the web

    Passive auto-targeting gives way to deliberate CTRL-to-sweep radar locking with a live target viewfinder and kill cam, the Shakedown Run tutorial teaches the first twelve minutes, typed damage meets per-section resistances, and the project gets a main menu and a landing site on the web.

  • 2026-07-10  //  v0.4.0

    Devlog #4: guided torpedoes, targeting and an enemy that fights back

    The flight toy becomes a combat game: proportionally-navigated guided torpedoes, a targeting arc with per-section fine-lock, turret auto-aim with true intercept lead, an AI wave with a real behavior state machine, a center-of-mass flight-assist overhaul, and the game's first audio and combat juice.

  • 2025-11-29  //  v0.3.0

    Devlog #3: zones, torpedoes and blast damage

    Lifecycle and zone-entry events for richer scenarios, the first area-of-effect weapon in the torpedo bay, a reworked per-section health model, and sharper directional and thruster shaders.

  • 2025-11-08  //  v0.2.0

    Devlog #2: objectives, enemy AI and better asteroids

    A data-driven events/filters/actions system for objectives, the first (gloriously dumb) enemy AI, procedural asteroids, and a week-long fight with GlobalTransform. Video included.

  • 2025-10-21  //  v0.1.0

    Devlog #1: modular ships and first combat

    Where it all started: thruster-driven modular ships, PD-controlled steering, turrets that shoot, and sections that shatter into chunks. Video included.

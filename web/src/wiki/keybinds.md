# Keybinds

The full control reference. Thrust is analog on the gamepad and on/off on the keyboard. The autopilot verbs each fly a whole maneuver and hand control back; any manual input disengages them immediately. New players should start with the [tutorial](../../tutorial/), which teaches these one at a time. The same bindings are viewable in-game under **Settings > Controls**, reachable from both the main menu and the pause menu.

## Weapons

<table class="controls">

            ></span>

        </td>
    </tr>
    <tr>
        <td>Launch torpedo</td>
        <td>
            <span
                class="pf pf-mouse-1"
                role="img"
                aria-label="Left mouse button"
            ></span>
            (default)
        </td>
    </tr>
</table>

Weapon sections are rebindable: in the editor, click a section to bind it to any key or mouse button, so a ship's turret and torpedo controls are whatever its build assigns. The shipped ship fires both turrets and torpedoes on the left mouse button; a torpedo only launches while you hold a raised (red) combat lock.

Keybinds

The full control reference. Thrust is analog on a gamepad trigger and on/off on the keyboard. The autopilot verbs each fly a whole maneuver and hand control back; any manual input disengages them immediately. New players should start with the tutorial, which teaches these one at a time.

Flight

Main thruster burn W / Space  or  RT
GOTO - fly to the current lock G  or  North button
ORBIT - park into orbit around a gravity well O  or  D-Pad Down
STOP - face retrograde and burn to rest X  or  East button
CANCEL autopilot (resume manual) Z  or  West button

Targeting and camera

Aim / look Mouse motion  or  Right stick
Free look Alt (hold)  or  LT
Raise weapons (combat stance) Right mouse button (hold)
Radar lock (hold to sweep) Ctrl (hold)  or  D-Pad Up
Clear lock (staged: combat, then nav) Ctrl (tap)  or  D-Pad Up (tap)
Cycle fine-lock component [ / ]  or  D-Pad Left / Right

Weapons

Fire turrets Left mouse button  or  RT2
Launch torpedo Raise weapons, hold a red combat lock, then fire the torpedo bay

Interface

Cycle HUD (All / Minimal / None) ` (grave / tilde)
Pause menu Esc
Back to editor (Sandbox only) F1
